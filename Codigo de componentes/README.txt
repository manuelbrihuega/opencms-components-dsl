Para que cada componente de la siguiente lista funcione, es necesario colocar los dos archivos de código, 
exactamente donde se indica a continuación:

- Banner: XSD -> /system/modules/es.dipucadiz2.basico/schemas/widgets/
	  JSP -> /system/modules/es.dipucadiz2.basico/formatters/widgets/

- Redes sociales: XSD -> /system/modules/es.dipucadiz2.basico/schemas/widgets/
		  JSP -> /system/modules/es.dipucadiz2.basico/formatters/widgets/

- Metacomponente: Contiene todos los elementos visuales y ocultos, servirá para extraer trozos de código.
	XSD -> /system/modules/es.dipucadiz2.basico/schemas/basicos/
	JSP -> /system/modules/es.dipucadiz2.basico/formatters/basicos/

- sivideo: Componente que se encargará de dibujar un video
	XSD -> /system/modules/es.dipucadiz2.basico/schemas/multimedia/
	JSP -> /system/modules/es.dipucadiz2.basico/formatters/multimedia/

- siaudio: Componente que se encargará de dibujar un audio
	XSD -> /system/modules/es.dipucadiz2.basico/schemas/multimedia/
	JSP -> /system/modules/es.dipucadiz2.basico/formatters/multimedia/
